## 3

```
我希望实现一个pygame程序，

当按下1、2、3、4、5、6、7、8、9、0的时候

会发出C4、D4、E4、F4、G4、A4、B4、C5、D5、E5的对应频率的声音

可以在初始化的时候先在piano_sounds目录生成这些声音对应的wav文件

如果按键抬起，则声音会停止。
```
